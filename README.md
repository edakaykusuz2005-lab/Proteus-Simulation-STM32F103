# Proteus Simulation - STM32F103

This repository contains the Proteus simulation files used during the development of the Automated Filter Frequency Response Measurement System.

Contents:
- Proteus project file (.pdsprj)
- STM32F103 firmware HEX file
- DDS based frequency sweep simulation

The simulation was used to verify communication between STM32F103 and AD9833 DDS signal generator before implementation on STM32F407 hardware.
